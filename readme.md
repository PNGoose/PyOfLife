A "Game of Life" made with python.
Preview:
[[preview.gif]]

Commands:
- pos **x/y/c** - place '#' or '.' on coordinates \[x y]
- nxco **nnn** - set the next command turn, by defaults it is after 10 turns
- fis - set the finish turn, when the game stops
- stop - stop the game manually
- cler - clear the board
- regen - set random board 
- help - show help message
